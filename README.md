# Secret Room Media — новостной портал (демка)

Дерзкое iGaming-медиа на фирменном стиле Secret Room. Статичная демо-версия: HTML/CSS/JS, открывается без сервера.

## Как запустить
- Просто открыть `index.html` в браузере, **или**
- Локальный сервер (чтобы работали пути и ИИ-проверка):
  ```bash
  python3 -m http.server 8099
  # → http://localhost:8099/index.html
  ```

## Разделы
- `index.html` — главная: основное полотно статей + сайдбар «из Telegram» + рекламные баннеры.
- `articles.html` — все статьи с сортировкой по тематике (чипы-фильтры, `?cat=`).
- `article.html?id=...` — страница статьи, считает просмотры.
- `calendar.html` — календарь iGaming внутри медиа.
- `services.html` — партнёрские сервисы и промики.
- `careers.html` — вакансии.
- `seo/*.html` — **скрытый SEO-блок**: в меню не выведен, но есть в `sitemap.xml` и в скрытых ссылках подвала → заходят из Google.
- `admin.html` — редакция (демо-пароль `secret2026`).

## Админка (как посты в соцсетях)
- **Новый пост** — композер в один экран: заголовок, текст (абзацы через пустую строку), тип (статья / из ТГ / реклама), тематика, эмодзи-обложка, цвет акцента.
- **SEO-метки** — простым языком, с подсказками, счётчиками (title ≤60, description ≤160), живым превью выдачи Google и кнопкой «Заполнить автоматически».
- **🤖 ИИ-проверка** — кнопка «Проверить текст ИИ»: OpenAI проверяет орфографию/пунктуацию/стиль и даёт короткие рекомендации, сохраняя дерзкий тон. Ключ вставляется во вкладке «Настройки» (хранится в браузере).
- **Статистика** — просмотры всего/сегодня, среднее на пост, топ материалов с барами.
- **Все посты** — список, правка/удаление.

## Данные
- Дефолтный контент — `js/data.js` (залиты реальные посты канала).
- Всё из админки — в `localStorage` браузера (`js/store.js`), подмешивается поверх дефолтных.

## Путь в прод
Демо на localStorage → миграция на **Supabase** (Postgres + Auth для нескольких редакторов + Storage под картинки + серверная статистика). ИИ-проверку в проде вынести на серверную функцию, чтобы ключ OpenAI не светился в браузере.

## Стек
Vanilla HTML/CSS/JS, шрифты Archivo + Inter (Google Fonts). Без сборки и зависимостей.
