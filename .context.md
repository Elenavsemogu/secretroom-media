## Описание
Демка новостного iGaming-портала для Secret Room Media. Дерзкий тон канала, фирменный стиль (светлый фон + акценты жёлтый/лайм/синий/розовый, логотип-скважина). Статичный сайт (HTML/CSS/JS, без сборки), статьи пока в localStorage; партнёры — в Supabase.

## Что сделано
- Дизайн-система, главная, статьи, статья, о нас, вакансии, SEO-блок, админка с верстаком и ИИ.
- **Календарь в медиа:** пункт меню + `calendar.html` (хлебные крошки, iframe `https://igaming-calendar.com/`, подвал). Сам calendar-2026 не правится.
- **Виджет на главной:** блок «Ивенты в этом месяце» (`js/calendar-widget.js`) — тянет `events.json`, клик → `calendar.html`.
- **Сервисы:** `services.html` + данные из Supabase `media_partners` (проект blackjack-royale). Правка в Table Editor; на сайте карточки по категориям, копирование промокодов, бейдж «Отличное предложение».
- WordPress — отложен.
