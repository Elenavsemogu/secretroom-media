# Проект: Secret Room Media — новостной портал

## Мета

- **Приоритет:** medium
- **Последнее обновление:** 2026-07-14

## Описание
Демка новостного iGaming-портала для Secret Room Media. Дерзкий тон канала, фирменный стиль (светлый фон + акценты жёлтый/лайм/синий/розовый, логотип-скважина). Статичный сайт (HTML/CSS/JS, без сборки), данные в localStorage. Референсы: Carbon (тёмный неон), BRVND «Strategy by Design» (брутальная типографика), структура — iGB.

## Что сделано
- Дизайн-система `css/styles.css`: светлый фон #FBF2E8, ink #1B1B1B, акценты #F5DA0F/#C8E712/#2E39F7/#F6ADE5, крупная типографика Archivo+Inter, логотип-скважина (SVG в common.js).
- **Главная** (`index.html`/`main.js`): hero + основное полотно + сайдбар «из Telegram» + рекламные баннеры.
- **Статьи** (`articles.html`): лента + сортировка по тематике (чипы).
- **Статья** (`article.html`): рендер + счётчик просмотров, промо/ТГ-плашки, inline-реклама.
- **О нас/Команда** (`about.html`), **Вакансии** (`careers.html`).
- **Скрытый SEO-блок** `seo/*.html` + `robots.txt` + `sitemap.xml` (не в меню, но индексируется, ссылки в скрытом подвале).
- **Админка** `admin.html`/`admin.js` (пароль `secret2026`): соцсеть-композер, SEO-поля с подсказками+превью SERP+автозаполнение, ИИ-проверка через OpenAI (ключ в настройках), статистика, список постов.
- Реальные посты канала залиты в `js/data.js` (новости / из ТГ / реклама).
