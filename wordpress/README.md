# Secret Room Media — WordPress (фаза 1)

Дубль публичного сайта в виде **темы WordPress**.  
Статический сайт на GitHub Pages **не трогаем** — он продолжает работать.

Кастомную админку (верстак, ИИ, Пульс) в эту фазу **не** переносим.  
Контент правите через обычную админку WordPress: `/wp-admin`.

---

## Что от вас нужно (простые шаги)

### 1. Хостинг с WordPress
Купите обычный хостинг с кнопкой «Установить WordPress» (Timeweb, Beget, Hostinger, Reg.ru и т.п.).

На панели хостинга:
1. Создайте сайт / домен (можно сначала технический адрес вида `xxx.hosting.ru`)
2. Нажмите **Установить WordPress**
3. Запомните логин и пароль админа

### 2. Скачайте тему
В этой папке проекта лежит тема:

`wordpress/secretroom-media/`

Соберите zip **из содержимого темы** (внутри zip сразу `style.css`, `functions.php`… — не лишняя обёртка).

Или возьмите готовый файл (если есть в релизе):

`wordpress/secretroom-media.zip`

### 3. Установите тему
1. Зайдите на `вашсайт.ru/wp-admin`
2. **Внешний вид → Темы → Добавить → Загрузить тему**
3. Выберите zip → Установить → **Активировать**

После активации тема сама создаст страницы: Главная, Статьи, Сервисы, Вакансии, Календарь.

### 4. Загрузите демо-контент
1. В админке: **Инструменты → Secret Room: демо**
2. Нажмите **Загрузить демо-контент**
3. Откройте сайт — должны появиться статьи, сервисы, вакансии

### 5. Проверьте
Откройте на сайте:
- главную
- статьи
- сервисы
- вакансии
- календарь

Если что-то криво — напишите мне, что именно.

## Как потом добавлять контент (без меня)

| Что хотите | Куда в админке |
|---|---|
| Новость / статья | **Записи → Добавить** (справа: формат main/tg/promo, акцент, эмодзи) |
| Сервис-партнёр | **Сервисы → Добавить** (+ категория справа, промокод, ссылка) |
| Вакансия | **Вакансии → Добавить** |
